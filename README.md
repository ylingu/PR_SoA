## 📚 项目简介
本项目包含宇航学院模式识别课程部分算法的实现以及实验代码。

## 🌍 运行环境

推荐环境：

- gcc 14.1.1+
- xmake 2.9.1+

关于xmake的安装请参考[官方文档](https://xmake.io/#/zh-cn/guide/installation)

## ⚡️ 快速开始

### <a name="1"></a>1. 安装

克隆本项目并进入项目根目录

```bash
git clone https://github.com/ylingu/PatternRecognition_SoA.git
cd PatternRecognition_SoA
```

### <a name="2"></a>2. 使用

运行实验代码

```bash
xmake
xmake run exp1
```
